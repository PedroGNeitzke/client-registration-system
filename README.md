# Client Registration System

A JavaScript project focused on DOM manipulation, dynamic rendering, form validation, and array management.

## Features

* Dynamic client cards
* Email validation
* Phone validation
* Duplicate checking
* Remove clients
* Real-time DOM updates

## Technologies

* HTML5
* CSS3
* JavaScript

## Purpose

This project was created to practice JavaScript fundamentals, especially DOM manipulation, events, arrays, validation, and dynamic rendering.

# Sistema de Cadastro de Clientes

Projeto em JavaScript focado em manipulação do DOM, renderização dinâmica, validação de formulários e gerenciamento de arrays.

## Funcionalidades

* Cards dinâmicos de clientes
* Validação de e-mail
* Validação de telefone
* Verificação de duplicados
* Remoção de clientes
* Atualização dinâmica do DOM

## Tecnologias

* HTML5
* CSS3
* JavaScript

## Objetivo

Este projeto foi criado para praticar fundamentos de JavaScript, especialmente manipulação do DOM, eventos, arrays, validação e renderização dinâmica.
